# Facture LaTeX

A simple LaTeX template for generating French invoices.

## Usage

1. Edit `main.tex` to customize your invoice details
2. Compile with LaTeX:
   ```bash
   pdflatex main.tex
   ```

## Features

- Professional invoice layout
- Automatic calculations
- Customizable line items with descriptions
- French format (Euro currency, date format)
